# Quick start

For the quick start example we're simply going to get some public beaches

<Steps>

```bash
npm install @kodverk/umea-open-api
```

```javascript
import { UmeaOpenAPI } from "@kodverk/umea-open-api"

const api = new UmeaOpenAPI()
const response = await api.publicBeaches.get()
```
