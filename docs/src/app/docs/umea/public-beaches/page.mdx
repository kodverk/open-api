# Public Beaches

This repo contains an node-wrapper for Ume√• municipality's [public beaches API](https://opendata.umea.se/explore/dataset/badplatser/api/).

## Usage

<Steps>

### Install dependencies
```bash
npm install -D @kodverket/umea-public-beaches
```

### Import API wrapper.
```javascript
import { UmeaOpenAPI } from "@kodverk/umea-open-api"
```

### Begin using the wrapper.
```javascript
const api = new UmeaOpenAPI()
const response = await api.publicBeaches.get()
```

</Steps>
